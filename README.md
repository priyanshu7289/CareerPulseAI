## 📌 Overview

CareerPulse AI is an AI-powered career intelligence platform that helps users analyze resumes, identify skill gaps, estimate salaries, explore career roadmaps, and receive personalized recommendations.

---

## ✨ Features

- 🔐 Secure Login & Registration
- 📄 Resume Analyzer
- 📈 Skill Gap Analysis
- 💰 Salary Predictor
- 💼 Job Listings
- 🧭 Career Roadmap
- 👤 User Profile & Settings
- 📊 Interactive Dashboard
- 🗄 PostgreSQL (Supabase)
- ☁ Streamlit Cloud Deployment
